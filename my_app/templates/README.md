サバヨミ (Sabayomi)

「締め切り直前までやる気が出ない」あなたへ贈る、行動変容カレンダー

・概要 (Overview)

「サバヨミ」は、先延ばし癖を技術でハックするためのタスク管理アプリケーションです。
ユーザーが入力した「本当の締め切り」を元に、システムが自動的に計算した「偽の締め切り（手前の日付）」を表示し、心理的なアンカリング効果によって早期着手を促します。



・解決する課題 (Problem)

- 課題: 「締め切り直前まで着手できない」という行動心理。自分で手帳に早い締め切りを書いても、「どうせ嘘だ」と自分を甘やかしてしまう。

既存アプリの限界: 高機能すぎるアプリは入力が面倒で続かず、単なるToDoリストでは強制力が足りない。

・ソリューション (Solution)

- 情報の完全隠蔽:

サーバーサイドで日付を計算し、クライアント（ユーザー）には「今見るべき日付」だけを渡します。自分自身をシステム的に騙すことができます。

- アメとムチの自動切り替え:

通常時 (Blue): 偽の期日を表示し、安心感を与えつつ誘導します。

緊急時 (Red): 偽の期日を過ぎると、即座に「本当の期日」に切り替わり、警告色でラストスパートを促します。

・使用技術 (Tech Stack)

- Backend: Python 3.x (Flask)

日付計算ロジック、APIエンドポイント

- Frontend: HTML5, CSS3, JavaScript (FullCalendar.js v6)

レスポンシブUI、非同期通信 (Fetch API)

- Design: スマホ・タブレット・PCのどの環境でも見やすいように調整

・こだわりポイント (Key Features)

1. サーバーサイドでのロジックブラックボックス化

JavaScriptで日付計算を行うとブラウザの開発者ツール等で「本当の日付」が見えてしまうため、計算ロジックは全てPythonサーバー側で実装し、API経由で表示文字列のみを送信する設計にしました。

2. 徹底的なレスポンシブ対応

「いつでもどこでも焦りを感じる」ために、スマホでの体験を重視。
JavaScriptで画面サイズと向き（縦長/横長）を検知し、カレンダーの高さやヘッダーのレイアウトを動的に最適化しています。

3. 思考コストゼロのUX

入力項目を「タスク名」と「期日」だけに絞りました。「いつやるか」のペース配分はシステムが肩代わりするため、ユーザーは入力するだけで管理から解放されます。
